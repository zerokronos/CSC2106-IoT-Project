# Documentation Index

This repository keeps implementation and experiment documentation only.
The final report is maintained externally in Google Docs.

## Files and Purpose

- `docs/topic-map.md`: MQTT topic namespace, naming rules, and topic inventory.
- `docs/message-schema.md`: Canonical message fields and payload formats for LoRa and MQTT.
- `docs/test-log.csv`: Structured experiment log template for performance and delivery results.
- `docs/figures/`: Local images/diagrams that may be referenced by docs or presentations.

## Mapping to Report Sections

- Problem setup / system overview: reference `docs/topic-map.md` and `docs/message-schema.md`.
- Implementation details: reference protocol choices and field definitions in `docs/message-schema.md`.
- Evaluation / experiments: populate and summarize `docs/test-log.csv`.
- Visuals: place architecture diagrams and experiment charts in `docs/figures/`.
